# 游戏盾 SDK

基于人工智能引擎的DDoS防护产品<br><br>
专为APP研发的防护产品，提供windows，安卓，IOS版本的SDK集成。满足抗D，防C,流量加密、防DNS劫持、防屏蔽等业务需求。（可独立私有化部署、可出售源码）

我们支持游戏盾的私有化独立部署方案，可将我方SDK、验证端、节点等全部程序部署至贵方环境，与我方其他用户完全隔离，从而确保贵方获得安全、稳定与隐私保护。私有化部署后，我们采用授权模式进行收费，不基于日活跃用户数、带宽使用量等指标设定套餐，授权后无任何功能或用量限制。
产品提供日常技术支持，并可为您方的技术人员提供技术培训，以满足贵方自行维护的需求。
